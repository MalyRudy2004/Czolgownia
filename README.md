## Guys building a tank

| ID | Requirement | Priority |
| :--- | :--- | :---: |
| **M01** | **Tank shall be remotely controlled** | **---** |
| S101 | Tank is a device with WiFi connection | High |
| S102 | User controls the tank with use of their smartphone via app | High |
| S103 | Server connecting tank and app is set on app's side (Hotspot) | High |
| S104 | User sees live streamed image from the Tank | High |
| **M02** | **Tank shall be able to move forward, backward and turn** | **---** |
| S201 | Tank is supplied with 2 independent motors, one each side | High |
| S202 | User can adjust the speed of the tank with joystick | Low |
| **M03** | **Tank shall have the ability to fire** | **---** |
| S301 | Tank has the air compressor/mechanism connected to the barrel | High |
| S302 | Tank fires tennis balls from the barrel | High |
| **M04** | **User shall be able to control the position of the barrel and turret** | **---** |
| S401 | Tank turret is placed on the motor rotating it horizontally | High |
| S402 | Tank has 2 sensors monitoring position of the turret (cable protection) | Low |
| S403 | Tank barrel is connected to the motor controlling its vertical angle | High |
| S404 | Tank has 2 sensors monitoring position of the barrel (range limit) | Low |
| **M05** | **Tank shall have an ammo storage and be able to reload** | **---** |
| S501 | Tank has a storage system controlled by the user | High |
| S502 | Feeder is sealed from the barrel not to lose pressure | High |
| S503 | Feeder is controlled with one motor to spare space | High |
| **M06** | **Tank should be visibly similar to the real tank** | **---** |
